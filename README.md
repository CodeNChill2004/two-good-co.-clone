# two-good-co.-clone

Two Good Co. Landing Page Clone
- A responsive front-end clone of the Two Good Co. landing page built using HTML, CSS, JavaScript, GSAP, and Locomotive Scroll.

- This project focuses on smooth scrolling effects, modern UI design, animations, hover interactions, and responsive layout inspired by the original Two Good Co. website.

🚀 Features :
- Smooth scrolling using Locomotive Scroll
- Scroll-based animations using GSAP + ScrollTrigger
- Custom cursor animation
- Interactive product cards with hover effects
- Responsive navigation bar animations
- Hero section loading animations
- Modern and clean landing page design
- Fully responsive layout

🛠️ Technologies Used :
- HTML5
- CSS3
- JavaScript (ES6)
- GSAP (GreenSock Animation Platform)
- Locomotive Scroll

   Through this project, I improved my skills in:
- Advanced CSS styling
- JavaScript DOM manipulation
- GSAP animations
- ScrollTrigger integration
- Locomotive Scroll implementation
- Creating smooth UI/UX interactions
- Front-end project structuring
